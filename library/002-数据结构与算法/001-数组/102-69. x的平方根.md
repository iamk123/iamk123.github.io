### [69. x 的平方根 ](https://leetcode.cn/problems/sqrtx/)

#### 方法：二分

```
class Solution {
    public int mySqrt(int x) {
        int l = 0;
        int r = x;
        int mid = 0;
        while(l <= r) {
            mid = l + ((r - l) >> 1);
            if((long)mid*mid == x) {
                return mid;
            } else if((long)mid*mid < x) {
                l = mid + 1;
            } else {
                r = mid - 1;
            }
        }

        return mid * mid > x ? mid - 1 : mid;
    }
}
```

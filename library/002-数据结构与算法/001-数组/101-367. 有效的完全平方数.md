### [367. 有效的完全平方数](https://leetcode.cn/problems/valid-perfect-square/)



```java
class Solution {
    public boolean isPerfectSquare(int num) {
        int l = 0; 
        int r = num;
        while(l <= r) {
            int mid = l + ((r - l) >> 1);
            long square = (long) mid * mid;
            if(square == num) {
                return true;
            } else if(square < num) {
                l = mid + 1;
            } else {
                r = mid - 1;
            }
        }

        return false;
    }
}
```

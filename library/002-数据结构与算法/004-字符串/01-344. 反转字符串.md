## 知识点

-   元素互换`s[l], s[r] = s[r], s[l]`



#### [344. 反转字符串](https://leetcode.cn/problems/reverse-string/)

##### 方法一：双指针

java

```java
class Solution {
    public void reverseString(char[] s) {
        int n = s.length;
        for (int left = 0, right = n - 1; left < right; ++left, --right) {
            char tmp = s[left];
            s[left] = s[right];
            s[right] = tmp;
        }
    }
}
```

go

```go
func reverseString(s []byte)  {
    l, r := 0, len(s) - 1
    for l < r {
        s[l], s[r] = s[r], s[l]
        l++
        r--
    }
}
```

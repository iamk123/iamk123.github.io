```
商用发动机健康管理平台

项目描述：
项目是与中国航发商发合作项目，旨在实时监控航空发动机试验，并能收集试验数据进行处理、分析及诊断，协助试验推进，实现数据分析全流程。开发了一个包含试验管理、状态监视、异常自动检测、故障诊断和数据可视化等功能的平台。

技术栈：
Springboot + Mybatis + MySQL + Redis + Mqtt; Python + Celery + HBase + MongoDB;

主要工作：
（1）参与系统设计，设计了数据源、数据集、实体模型、实体等，解决多类型结构数据存储、衍生和溯源问题。
（2）设计工作流引擎基础架构，包含流程构建器（任务节点、逻辑类节点、事件类节点等）、解析器、执行器、表达式解析器、监视器等。使用线程池、MQTT、Celery实现了一个分布式工作流执行引擎。
（3）负责算法模型的设计，实现了算法模型管理、分布式单算法任务管理；并参与部分算法模型的开发，包括指定维度排序、缺失值处理、台阶计算等算法。
（4）通过MinIO实现非结构化数据管理，并通过文件分片上传、断点续传、文件秒传等优化大文件上传过程；通过分布式锁解决文件夹重命名、复制、删除问题
（5）使用MQTT构建异步消息系统，实现发送通知、异常告警、任务状态同步和工作流任务推进。

项目优化：
（1）通过将业务流程从串行改为并行处理，提高接口响应速度，将耗时减少超过一半。
（2）通过优化数据库索引、SQL语句等优化慢SQL问题，如limit大分页问题、索引失效问题。
```

```
weTeam校园

项目描述：
项目是一个校园活动交流平台，旨在解决疫情期间活动查找难、宣传难及组队难等问题。实现了注册登录、活动管理、组队管理、点赞评论、消息提醒和网站数据统计等功能。项目获中国高校计算机大赛西北赛区一等奖、全国三等奖，成功申报国家级创新创业项目，并以该项目作为创业项目基石继续发展。线上地址:微信小程序"团团小文体"

技术栈：
SpringBoot + MyBatis + MySQL + Redis + Kafka

工作内容： 
（1）实现了活动管理、组队管理、评论管理等模块。
（2）使用JWT+Redis实现登录功能，解决分布式Session问题，采用双令牌认证的方式实现用户无感知登录刷新。
（3）使用Redis实现热门榜单、点赞关注收藏等，并解决数据库与缓存双写一致性问题、缓存穿透、缓存雪崩、缓存击穿问题。
（4）使用Kafka构建异步消息系统，发送评论、点赞、关注和活动更新等通知，起到对系统削峰、解耦和异步调用的作用。
（5）使用AOP记录系统日志、权限检查，对项目进行解耦，增加重用性和和可维护性。

项目收获：
（1）以竞赛为驱动，短期内学习SpringBoot、MyBatis、Redis等技术，完成一个完整的前后端分离的项目。通过项目了解Redis、Kafka在项目中的具体应用。
（2）学习了Docker基本用法，通过容器来部署项目前后端。
```

成就感最大的事情

```
就是在大学期间参加的一个竞赛，也就是简历中的weTeam校园这个项目，当时正值新馆疫情期间，曾经有一段很长的时间封锁在宿舍里，这时候我就发现，学校中举办的一些活动都取消了线下宣讲，只能在线上进行宣传，这些信息也都比较杂乱无章，缺少一个统一的管理平台，并且我们想要找一些队友参加比赛的话也只能在各种群里问，这些信息很快就会被各种信息所覆盖掉，造成找人组队参赛比较困难。所以基于这些问题，我设计了这个系统。
但是当时没有志同道合的人一起去完成，我就只能自己一个人完成，而比赛的截止日期也很短，不到一个月时间，此外我对开发这个系统要用到各种技术还不熟，所以压力很大，也不知道能否如期完成。
所以之后就是处于一种边学习边开发的状态，白天上课，课余时间就开发系统，经过努力，还是在比赛前完成了系统，并且上线使用。令人高兴的是，由于项目的点子很好，实现的功能也比较完善，评委老师给了很高的评价，并且帮助我在学院里面推广使用，最后拿到了比赛的一等奖

而我也通过自己的平台，找到了一些志同道合的队友，一起对系统进行完善改进，并一起参加之后的中国高校计算机大赛全国，并获得了西北赛区一等奖和全国三等奖的成绩。
这就是我成就感最大的事情。
```
